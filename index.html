SAC

SAC is a project that uses information from Steam. This information requires the user to provide either a user id or custom url and an appid. If the user provides these they will be able to see all the remaining achievements, that the user still needs to get for that specific game.

Character Generator

Character Generator is a project that generates a character, which you might already have guessed. The character needs the user to enter/select a few details about the character. After the user has done that, they can click a button which will generate the character, based on the information provided.
